# 🌤️ Weather App

A responsive weather app built with React.js and Tailwind CSS.

## 🔗 Live Demo
[weather-app-ayushi.vercel.app](your-link-here)

## ✨ Features
- Search weather by city name
- 5-day forecast
- Auto-detect location
- Celsius/Fahrenheit toggle
- Responsive on all devices

## 🛠️ Tech Stack
React.js | Tailwind CSS | OpenWeatherMap API

## 🚀 Run Locally
1. Clone the repo
2. Create .env file with REACT_APP_WEATHER_API_KEY=your_key
3. npm install && npm start
